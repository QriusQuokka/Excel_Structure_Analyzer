# Excel Structure Analyzer

처음 받아본, 설명해 줄 사람도 없는 다중 시트 엑셀 파일의 **구조와 시트 간 수식 관계**를
자동으로 파악해 HTML 리포트로 보여주는 도구입니다.

- 모든 시트 목록과 크기·수식 개수 파악
- 시트별 수식 추출
- 수식 안의 시트 참조(`SheetB!A1`, `'시트 이름'!A1`, 3D 참조 `A:C!A1` 등)를 분석해
  **시트 간 데이터 흐름(입력 → 가공 → 산출)** 을 그래프로 시각화
- **입력단/산출단 시트 자동 식별**

## 설치

Python 3.10+ 와 openpyxl 만 있으면 됩니다.

```bash
pip install openpyxl
```

## 사용법

```bash
python analyze.py 파일.xlsx
```

옵션:

| 옵션 | 설명 |
|------|------|
| `-o, --output 경로` | 출력 HTML 경로 (기본: `입력파일명_analysis.html`) |
| `--no-open` | 분석 후 브라우저를 자동으로 열지 않음 |

실행하면 HTML 리포트가 생성되고 브라우저가 자동으로 열립니다.

## 리포트 보는 법

- **관계도**: 화살표는 데이터 흐름 방향입니다. `A → B` 는 "B가 A의 값을 참조한다"는 뜻.
  화살표 위 숫자는 참조하는 셀의 개수입니다.
  - 파란색 = **입력단**(다른 시트를 참조하지 않는 시트)
  - 초록색 = **산출단**(아무 시트도 참조하지 않는 시트, 보통 최종 결과)
- **시트 목록**: 각 시트의 크기, 값/수식 셀 개수, 역할 태그
- **관계 상세**: 펼치면 "어느 시트의 어느 셀이 어떤 수식으로 어디를 참조하는지" 확인 가능

## ⚠️ 구버전 .xls 파일

이 도구는 `.xlsx` / `.xlsm` 만 분석합니다.
오래된 **`.xls` 파일은 엑셀에서 열어 [다른 이름으로 저장] → `Excel 통합 문서(*.xlsx)`** 로
다시 저장한 뒤 그 파일을 분석해 주세요. (.xls 를 넣으면 같은 안내 메시지가 나옵니다.)

## 한계

- 시트 단위 관계에 초점을 맞춥니다(셀 단위 추적은 "관계 상세"에서 샘플로 확인).
- 외부 통합문서(`[1]Sheet!A1`)나 이름 정의(Named Range)를 통한 참조는 관계도에 표시되지
  않으며, 외부 참조 개수는 리포트 하단 "참고"에 별도 표시됩니다.

## 구조

```
excel_analyzer/
  formula_parser.py   수식에서 시트 참조 추출 (정규식 + 실제 시트명 대조)
  analyzer.py         openpyxl 로 시트/수식/의존관계 분석
  report.py           HTML + Mermaid 리포트 렌더링
analyze.py            CLI 진입점
make_sample.py        검증용 샘플 통합문서 생성
```
